The Vehicle Rental System is a C# application designed to manage the rental of vehicles such as cars, motorcycles, and cargo vans. It utilizes object-oriented programming concepts, specifically inheritance, to model different types of vehicles and their rental operations. The system allows customers to rent vehicles, return them, and automatically generate an invoice upon return using predefined test data.

Vehicle Types: 
The system supports three types of vehicles: Car, Motorcycle, Cargo Van 

Rental Operations:
Vehicle can be rented by specifying the custamer and rental duration.
Return a rented vehicle, which triggers the generation of an invoice.
View rental history and invoice details.
Invoice Generation: Upon vehicle return, an invoice is automatically generated with details including rental duration, rental cost, and any additional charges.

Test Data: Predefined test data is used to simulate rental operations and demonstrate invoice generation.
